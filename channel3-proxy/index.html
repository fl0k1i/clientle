How to use your key

Search products
curl -X POST http://localhost:8000/search \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{"query": "nike shoes", "limit": 10}'
In stock only
curl -X POST http://localhost:8000/search \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{"query": "shoes", "filters": {"availability": ["InStock"]}}'
Product detail
curl http://localhost:8000/products/PRODUCT_ID \
  -H "x-api-key: YOUR_API_KEY"
Similar products
curl http://localhost:8000/products/PRODUCT_ID/similar \
  -H "x-api-key: YOUR_API_KEY"
Lookup by URL
curl "http://localhost:8000/lookup?url=https://nike.com/..." \
  -H "x-api-key: YOUR_API_KEY"
